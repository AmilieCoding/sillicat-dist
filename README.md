See the guide, here.

# Do this after running setup.sh and opening the new project in Intellij.
Set `-Djava.library.path=./test_natives` as your VM option.
Set java to temurin-1.8
Set your main method class to Start (default package)
Set your working directory to have /test_run/ on the end.
